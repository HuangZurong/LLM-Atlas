# NLP Fundamentals

The essence of NLP is **enabling computers to understand, process, and generate human language**. This module serves as the prerequisite foundation for the LLM knowledge system, tracing the core arc of NLP as a discipline — from linguistic theory to the arrival of the Transformer.

> This module is a concise NLP overview. From [02_Scientist](../02_Scientist/) onward, the content dives into LLM specifics.

## Curriculum

### Layer 1: Linguistic Foundations — The "Soul" of NLP

### Layer 2: Classical NLP & The Statistical Era

2. **[Text Preprocessing](./02_Classical_NLP/01_Text_Preprocessing.md)**: Tokenization, text normalization, stop words, regular expressions.

4. **[Statistical Models](./02_Classical_NLP/03_Statistical_Models.md)**: Naive Bayes, HMM, CRF.

### Layer 3: The Deep Learning Era — From Discrete Symbols to Continuous Vectors
